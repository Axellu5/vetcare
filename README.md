# VetCare - Veterinarijos Klinikos Sistema

## Tikslas

VetCare - vidinė veterinarijos klinikos valdymo sistema, skirta mažoms ir vidutinėms veterinarijos klinikoms. Sistema leidžia registruoti gyvūnus, jų savininkus, veterinarus bei klinikos vizitus, peržiūrėti gydymo istoriją, valdyti registracijas su kalendoriumi ir laiko slotais, bei atlikti paiešką, filtravimą ir rūšiavimą. Prieiga prie sistemos apsaugota administratoriaus prisijungimu.

## Technologijos

- Next.js 14 (App Router) - full-stack web karkasas
- React + Tailwind CSS - naudotojo sąsaja
- Supabase (PostgreSQL) - duomenų bazė
- Prisma ORM - duomenų bazės valdymas
- bcrypt + JWT - autentifikacija
- Jest - testavimas
- Git + GitHub - versijų kontrolė
