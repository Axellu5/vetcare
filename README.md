# VetCare - Veterinarijos Klinikos Sistema

## Tikslas

VetCare – vidinė veterinarijos klinikos valdymo sistema, skirta mažoms ir vidutinėms veterinarijos klinikoms. Sistema leidžia registruoti gyvūnus, jų savininkus, veterinarus bei klinikos vizitus, peržiūrėti gydymo istoriją, atlikti paiešką, filtravimą ir rūšiavimą.

## Technologijos

- **Next.js 14 (App Router)** — full-stack web karkasas
- **React + Tailwind CSS** — naudotojo sąsaja
- **Supabase (PostgreSQL)** — duomenų bazė
- **Prisma ORM** — duomenų bazės valdymas
- **Jest** — testavimas
- **Git + GitHub** — versijų kontrolė
